```
RuntimeError: expected scalar type Long but found Int
```

Dieser Fehlercode weist darauf hin, dass der Input im falschen tensor.type ist. Somit kamen wir nach langem Recherchieren auf diese Line Code, die das Problem gelöst hat.

```
labels = labels.type(to.LongTensor)
```

Labels ist hierbei die Variabel für den Input und der Rest ändert den tensor.type, also wie der tensor hinterlegt ist und aussieht.
